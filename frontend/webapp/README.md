# ⚔️ SkillVerse Frontend

Telegram Mini App frontend for the SkillVerse competitive gaming platform.

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Telegram Web App SDK

## Structure

```
frontend/webapp/
├── index.html      # Main HTML structure
├── style.css       # Styles and animations
├── script.js       # Modular JavaScript app
└── assets/         # Images and icons
```

## Screens

- Splash Screen
- Landing Page
- Dashboard
- Game Selection (RPS)
- Match Gameplay
- Victory/Defeat Screen
- Leaderboards
- Wallet
- Warrior Profile
- Daily Rewards

## Development

1. Open `index.html` in a browser
2. For Telegram Mini App testing, use ngrok or similar tunneling service
3. Configure the webapp URL in your Telegram Bot settings

## Build

No build step required. This is a static webapp that can be deployed to any static hosting service (Vercel, Netlify, etc.).
